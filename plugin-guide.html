iGaia Plugin Development Guide

Version 1.1 — Updated for built-in Theme Editor + Plugin Manager

⚠️ Security Note: Plugins run with full access to the renderer process. Only install plugins from trusted sources.

1. Installing Plugins

  1. Run iGaia at least once to create the plugins folder.
  2. Open the plugins folder:
    %APPDATA%\iGaia\plugins\
  3. Create a new folder for your plugin (e.g. my-cool-plugin).
  4. Inside that folder, add these two files:
    • manifest.json
    • index.js
  5. Restart iGaia — your plugin will load automatically.

2. Plugin Folder Structure

%APPDATA%\iGaia\plugins\
└── my-cool-plugin\
    ├── manifest.json
    └── index.js

3. manifest.json Example

```json { "name": "My Cool Plugin", "description": "Adds useful features to iGaia", "version": "1.0.0", "author": "Your Name" } ```

4. index.js – Basic Plugin Template

```javascript module.exports = { init: (api, manifest) => { console.log(`[${manifest.name}] Plugin loaded successfully!`); // Add a button to the toolbar api.addToolbarButton({ label: "🚀 My Feature" }); } }; ```

5. Available Plugin API (Current)

| Method | Description | Example | |-------------------------------|--------------------------------------------------|--------| | `api.addToolbarButton(options)` | Add button to toolbar | `api.addToolbarButton({ label: "🔥 Test" })` | | `api.injectScript(code)` | Run JS on the Gaia page | `api.injectScript('alert("Hello!")')` | | `api.navigateTo(url)` | Navigate to a URL | `api.navigateTo('https://www.gaiaonline.com/')` | | `api.reload()` | Reload current page | `api.reload()` | | `api.log(msg)` | Print to console | `api.log('Something happened')` | | `api.alert(msg)` | Show popup message | `api.alert('Hello from plugin!')` |

6. Full Example Plugin

```javascript module.exports = { init: (api) => { api.addToolbarButton({ label: "🔄 Quick Refresh" }); api.injectScript(` console.log('%c[Quick Refresh] Plugin active', 'color: lime'); `); } }; ```

7. Important Notes - Plugins are loaded from `%APPDATA%\iGaia\plugins\` - You do **not** need to rebuild the .exe when adding or editing plugins. - The built-in Theme Editor and Plugin Manager are now part of the core application. - For advanced features (custom panels, data saving, etc.), use `api.showPanel()` in future updates. - Restarting iGaia is currently required to fully enable/disable plugins.

8. Quick Access

Press Win + R and paste:
