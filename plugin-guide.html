iGaia Plugin Development Guide

Version 1.0 — For iGaia Native Flash Client

⚠️ Important: Plugins have full access to the renderer process. Only install plugins from sources you trust.

1. How to Install a Plugin

  1. Run iGaia at least once (it creates the necessary folders).
  2. Open the plugins folder:
    %APPDATA%\iGaia\plugins\
  3. Create a new folder for your plugin (example: my-awesome-plugin).
  4. Inside that folder, create two files:
    • manifest.json
    • index.js
  5. Restart iGaia — your plugin will load automatically.

2. Plugin Folder Structure

iGaia/
└── plugins/
    └── my-awesome-plugin/
        ├── manifest.json
        └── index.js

3. manifest.json

```json { "name": "My Awesome Plugin", "description": "Adds cool features to iGaia", "version": "1.0.0", "author": "Your Username" } ```

4. index.js – Basic Plugin Template

```javascript module.exports = { init: (api, manifest) => { console.log(`[${manifest.name}] Plugin loaded!`); // Add a button to the toolbar api.addToolbarButton({ label: "🚀 My Button" }); } }; ```

5. Available API Methods

MethodDescriptionExample
api.addToolbarButton(options)Add a button to the toolbarapi.addToolbarButton({ label: "⚙️ Settings" })
api.injectScript(code)Run JavaScript on the Gaia pageapi.injectScript('alert("Hello!")')
api.navigateTo(url)Navigate to a URLapi.navigateTo('https://www.gaiaonline.com/')
api.reload()Reload current pageapi.reload()
api.log(msg)Print to consoleapi.log('Hello from plugin')
api.alert(msg)Show popup messageapi.alert('Plugin message')

6. Full Example: Auto Refresh Button

```javascript module.exports = { init: (api) => { api.addToolbarButton({ label: "🔄 Refresh Gaia" }); api.injectScript(` console.log('%c[Auto Refresh] Plugin active', 'color: lime'); `); } }; ```

7. Quick Access to Plugins Folder

Press Win + R, paste the following, and press Enter:

%APPDATA%\iGaia\plugins
