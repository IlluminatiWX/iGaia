iGaia Plugin Development Guide

Version 1.1 — For iGaia Native Flash Client

⚠️ Important: Plugins have full access to the renderer process. Only install plugins from trusted sources.

1. How to Install a Plugin

  1. Run iGaia at least once (it creates the plugins folder).
  2. Open this folder on your PC:
    %APPDATA%\iGaia\plugins\
  3. Create a new folder for your plugin (e.g. my-cool-plugin).
  4. Inside that folder, create two files:
    • manifest.json
    • index.js
  5. Restart iGaia. Your plugin will load automatically.

2. Plugin Structure

%APPDATA%\iGaia\plugins\
└── my-cool-plugin\
    ├── manifest.json
    └── index.js

3. manifest.json Example

{
  "name": "My Cool Plugin",
  "description": "Adds useful features to iGaia",
  "version": "1.0.0",
  "author": "Your Name"
}

4. index.js – Basic Template

module.exports = {
  init: (api, manifest) => {
    console.log(`[${manifest.name}] Plugin loaded!`);

    api.addToolbarButton({
      label: "🚀 My Button"
    });
  }
};

5. Available API Methods

MethodDescription
api.addToolbarButton({ label })Add button to toolbar
api.injectScript(code)Run JavaScript on Gaia page
api.navigateTo(url)Navigate to URL
api.reload()Reload current page
api.log(msg)Print to console
api.alert(msg)Show popup

6. Quick Access to Plugins Folder

Press Win + R, paste this, and press Enter:

%APPDATA%\iGaia\plugins
